

1. Modify HTML (if necessary):

Ensure that you only use the popover attribute on the .contact-container. Remove any popover related attibutes on the .contact-popover div.


2. Modify CSS (styles.css):

Remove styles for .contact-popover:popover-open: You need to remove the styles you have that create the full-screen opaque background for .contact-popover when it's open, because we'll be handling the background blur with ::backdrop instead. Remove all styles in the styles.css from line 518 to line 527:
.contact-popover:popover-open {
  position: fixed;
  top: 0;
  left: 0;
  width: 100vw;
  height: 100vh;
  background-color: rgba(0, 0, 0, 0.7); /* Semi-transparent black */
  z-index: 999; /* Above everything else */
  opacity: 1;
}

Add styles for ::backdrop: Add the following CSS rules to create the backdrop blur effect:
/* Existing Popover Styles */
:popover-open {
    background-color: hsl(25, 100%, 50%);
    border: 0;
    color: hsl(300, 50%, 3%, 0.85);
    padding: 1.5rem;
    /* Start small and expand */
    height: 0;
    width: 0;
    overflow: hidden; /* Prevents content from showing before animation */
    transition: max-height 2s ease-in-out, max-width 2s ease-in-out, opacity 2s ease-in-out;
    opacity: 0;
}

/* New Backdrop Styles */
[popover]::backdrop {
    background-color: rgba(0, 0, 0, 0.5); /* Semi-transparent black */
    backdrop-filter: blur(5px); /* Add blur effect */
    -webkit-backdrop-filter: blur(5px); /* Safari compatibility */
}


Adjust .contact-container styles as needed. You don't have to change much here, as most of the styles look good. However, you need to make sure the .contact-container has the correct z-index and you can make sure its position is relative, as it will be relative to the popover.
.contact-container {
	display:flex;
	flex-direction:column;
	position: relative;
	top: 50%;
	left: 50%;
	transform: translate(-50%, -50%);
	background-color: white;
	border-radius: 30px 5px 30px 30px;
	box-shadow: 0px 4px 6px rgba(0, 0, 0, 0.1);
	width: 80%;
	max-width: 1200px;
	padding: 20px;
	z-index: 1000; /* Above the overlay */
}


Adjust the :popover-open styles to adjust the popover appearance. You may want to change the background-color for example.


3. Modify JavaScript (script.js):

Make sure that the modalClose will hide the popover of the .contact-container :
// When clicking on the element assigned to modalClosed, hide the open popover status to the div with the popover attribute, in this case, the div with the class .contact-container
modalClose.addEventListener("click", function () {
  document.querySelector(".contact-container").hidePopover();
});


Explanation of Changes:

::backdrop: This is a pseudo-element that allows you to style the area behind an element that's displayed in a "top layer," such as a <dialog> or an element with popover. In this case, you're applying the background color and blur effect directly to the backdrop, which is exactly what you want.
Removed.contact-popover:popover-open styles: Since the backdrop is now handling the overlay effect, the .contact-popover:popover-open styles that were creating that overlay are unnecessary and are removed.
modalClose: Now, the modalClose element will hide the popover of the .contact-container
